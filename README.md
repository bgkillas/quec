## keybindings
'i' to enter edit mode

'esc' to exit edit mode

'd' to cut line

'p' to print line

'w' to save

'q' to quit

'u'/'z' to undo

'y'/'x' to redo

'/' to start search mode

search mode:

'esc' to quit search mode

'enter' to search through file
